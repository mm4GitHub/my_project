# Analysis of DVD Rental

## Overview
Analysis of movies rental using PostgreSQL

## Tools Used
-PostgreSQL
-PgAdmin4

## Database Structure
- 15 tables including customer, film, rental, payment
- Sample data for practice and analysis

## Files
### Database Files
- restore.sql - Main restore script
- toc.dat - Database table of contents
- *.dat - Individual table data files

## Scripts
-dvdrental_explore_data.sql

## How to Restore Database in pgAdmin4
1. Open pgAdmin4
2. Right click your database
3. Click Restore
4. Select restore.sql file
5. Click Restore
